# Credit Dispute Metro 2 Generator

Generates credit bureau-ready Metro 2 dispute files from a simple form.

## For Bank Employees

### How to Use

1. **Double-click** `bank_app.py` (or run `python3 bank_app.py` in terminal)

2. **Fill out the form** with customer information

3. **Click "Add Another Dispute"** for each customer

4. **Click "Generate Metro 2 File"** when done

5. **Upload the generated `.dat` file** to the credit bureau portal
